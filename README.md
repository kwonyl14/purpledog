# Description

퍼플독 채용 백엔드 엔지니어 과제입니다.



# Getting Started

### Version Info

Spring Boot 2.6.4  

gradle 7.4.1  

mysql 8.0.28  

### OS

Windows 10  

### Prerequisite

DB는 개인 EC2에 실행중인 mysql 서버로 연결이 되어있어 바로 실행 가능합니다.  

실행 방법 :  

프로젝트를 clone받고 콘솔 실행 후 jar파일이 있는 경로로 이동 한 다음 아래 명령어 입력합니다.  

java -jar jar파일명.jar
